# RoadMap

Road Map of ElephantIM

## 1、端到端全局

### 1.1、功能列表

- 账号
  - 注册 Register
  - 登陆 Login
  - 登出 Logout
- 联系人
- 会话
  - 单聊
  - 群聊
  - Thread话题（进阶）
  - Channel频道（进阶）
    - 参考Telegram：
    - 只有Owner能发消息，其它人只读，支持超过20万成员
- 消息
  - 文字
  - 文件
    - 图片
    - 语音
    - 视频
  - 特殊消息（进阶）
    - 回复
    - 提到@
    - 表情 👍 👎 👌🏻
    - Markdown格式支持
